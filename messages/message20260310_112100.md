**FPS note:** CUDA gives same total FPS as CPU (~40 fps). PyBullet physics is CPU-only;
CUDA only accelerates SAC gradient updates, which are hidden in env-step time. Not worth
switching back — GPU is doing real work, just not the bottleneck.

3 conv layers). SmallCNN (k=4s2→k=3s2→k=3s1) added to train_vision_nav.py, selected
automatically when img_h<36 or img_w<36.
